## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/ticketwave.git

# Navigate to project directory
cd ticketwave

# Install dependencies
npm install

# Start development server

# Deploy contracts

```
